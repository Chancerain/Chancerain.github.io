数据类型

JS中一共分成六种数据类型

  • String 字符串
  • Number 数值
  • Boolean 布尔值
  • Null 空值
  • Undefined 未定义
  • Object 对象

基本数据类型

  • String

    在字符串中使用\作为转义字符
    ' ==> ‘
    " ==> “
    \n ==> 换行
    \t ==> 制表符
    \ ==> \

  • Number

    特殊的数字
    Infinity 正无穷
    Infinity 负无穷
    NaN 非法数字（Not A Number）
    其他进制的数字的表示：
    0b 开头表示二进制，但是不是所有的浏览器都支持
    0 开头表示八进制
    0x 开头表示十六进制

  • Boolean

  • Null

  • Undefined

类型转换

转换为String

  • 方法一 (强制转换)

    toString()

    var a = 123;
a = a.toString();

    这个方法不适用于null和undefined，由于这两个类型的数据中没有方法，所以调用toString()时会报错

  • 方法二 (强制转换)

    String()

    var a = 123;
a = String(a);

    对于Number Boolean String都会调用toString()方法来将其转换为字符串，对于null值，直接转换为字符串”null”。对于undefined直接转换为字符串”undefined”

  • 方法三 (隐式转换)

    任意的数据类型 +””

    var a = true;
a = a + "";

    和String()函数一样

转换为Number

  • 方法一 (强制转换)

    var s = "123";
s = Number(s);

    转换的情况：

    1. 字符串 –> 数字
      • 如果字符串是一个合法的数字，则直接转换为对应的数字
      • 如果字符串是一个非法的数字，则转换为NaN
      • 如果是一个空串或纯空格的字符串，则转换为0
    2. 布尔值 –> 数字
      • true转换为1
      • false转换为0
    3. 空值 –> 数字
      • null转换为0
    4. 未定义 –> 数字
      • undefined 转换为NaN

  • 方法二 (强制转换)

    parseInt()或parseFloat()

    • parseInt()可以将一个字符串中的有效的整数位提取出来，并转换为Number

      var a = "123.456px";
a = parseInt(a); //123

    • parseFloat()可以将一个字符串中的有效的小数位提取出来，并转换为Number

      var a = "123.456px";
a = parseFloat(a); //123.456

  • 方法三 (隐式转换)

    var a = "123";
a = +a;

    和Number()函数一样

转换为Boolean

  • 方法一 (强制转换)

    使用Boolean()函数

    var s = "false";
s = Boolean(s); //true

    转换的情况

    1. 字符串 –> 布尔

      除了空串其余全是true

    2. 数值 –> 布尔

      除了0和NaN其余的全是true

    3. null、undefined —> 布尔

      都是false

    4. 对象 —> 布尔

      都是true

  • 方法二 (隐式转换)

    为任意的数据类型做两次非运算，即可将其转换为布尔值

    var a = "hello";
a = !!a; //true
