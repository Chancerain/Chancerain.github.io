自我介绍

各位老师，你们好！很荣幸可以进入中南林业科技大学的研究生复试。我来自湖南长沙，23岁，本科专业是计算机科学与技术。接下来我将从三个方面进行自我介绍。

学业上，本科四年我系统的学习了计算机相关知识，有良好的专业素养和实操能力。获得过奖学金，也通过了英语六级考试。

专业技能方面，我对编程有浓厚的兴趣，自己写过一些项目，有开发经验。对C、Java等高级语言都有基本的掌握，主要的技术栈在前端开发方向，对JS、JQuery、Vue.js和服务端的Node.js等技术都熟练掌握，PC端、移动端、微信小程序端的项目都写过。也搭建部署了自己的个人技术博客，会经常在博客上产出一些技术方面的学习笔记，平时也经常逛掘金、思否之类的IT论坛。为了提高我的实操能力，我也有投简历在企业实习过开发岗位。实习期间我主要参与开发了一个CRM客户关系管理系统的项目，学习到了企业的代码编写规范、真实业务开发流程。我也将我熟悉技术栈和开发经验应用到了之后的毕业设计上，我的毕业设计是一个小区疫情防控平台的小程序，前后端都是用我熟悉的JS语言写的。

性格上，我也比较随和。我担任过校合唱团的声部长，无论是在课外组织活动还是实习的项目组里，我都能很好的和他人沟通交流合作，一起完成组内的任务。

本科的学习过程也让我认识到了自身的不足。由于缺少更深层次的知识，我无法从事更高层面的工作。我想通过研究生阶段的学习，接触到更前沿的技术，进一步提升自己的能力，所以选择考研继续深造。

以上是我的自我介绍，感谢老师们的聆听。

1. 毕业设计

我的毕业设计的题目是《基于uni-app的小区疫情防控平台的设计与实现》

技术栈：两个子系统，客户端是基于uni-app开发的微信小程序，后台管理端是基于Vue.js的H5页面，服务端是Node.js结合Express框架搭建的接口，数据库是MySQL。

主要功能：主要实现居民健康打卡、实时全国地区疫情数据、疫苗核酸预约、来访登记，管理员管理居民、发布核酸疫苗信息、ECharts小区数据可视化等功能。

开发流程：

  • 数据库：设计 - 建表
  • 服务端：搭建环境 - db包连接数据库 - 配置cors等中间件 - 编写后端路由 - 调试API - 生成接口文档
  • 前端：设计页面 - 画页面 - 调用接口
  • 测试 - 改bug

遇到的问题：

  1. uni-app转译成微信小程序，有些语法有兼容性问题，要换成特定的写法。

实习

实习期间主要参与开发了一个CRM客户关系管理系统。

开始看公司代码。

编码规范：

  1. 命名
  2. Vue项目文件/组件/插件 目录高度抽离封装
  3. 代码格式化
  4. Vuex统一状态管理
  5. 代码注释

项目介绍：嘉创CRM系统，一个基于uni-app，钉钉企业内置应用。主要功能是对客户、联系人、商机、跟进、回款、合同做一个统一管理。有用户、客户、商机、跟进、公海池等模块。（我负责其中的客户、联系人、商机模块）

技术栈：uni-app、uni-ui、Java、MySQL

开发流程：

  1. 项目启动大会
  2. 确定需求
  3. 没接口对照UI写页面
  4. 有接口后对照接口文档调接口
  5. 自测
  6. 前后端联调+改bug
  7. 更新需求
  8. 上线。

遇到的问题：

  1. 第一次自己写项目
  2. 上传图片问题

2. 学过的五门课

C语言：C语言是一种面向过程的编程语言，它被广泛应用于系统编程、嵌入式系统等领域，还是许多编程语言的底层核心。这门课主要学习了C语言的基本语法、数据类型、数组、指针、函数、结构体、文件等知识点。

计算机网络：计算机网络是一门介绍计算机网络结构、协议、技术和应用的课程。学习了计算机网络的基本原理，了解计算机网络在实际应用中的运作方式，同时学习网络的安全和管理等方面的基本知识。

软件工程：这门课程旨在介绍软件开发过程的不同阶段，如需求分析、设计、编写、测试和维护。学习如何在软件项目中应用不同的技术和工具，并了解编写高质量代码。

数据库原理：这门课介绍了数据库管理系统的设计与实现，包括数据模型，数据结构以及查询语言等方面。学习了数据库系统的原理以及数据库设计的基本规则。

数据结构：这门课包含三方面的内容，逻辑关系、存储关系及操作。学习了一系列常见的数据结构的逻辑结构、存储结构和查找、插入、删除等操作，还学习了一些常见的查找、排序算法。

4. 人工智能和互联网+的看法 / 前沿技术

人工智能是一种模拟人类智能的技术和应用，在没有明确的指令或者人的干预下，AI可以自主学习、推理、规划、感知、认知和决策。人工智能技术包括机器学习、自然语言处理和计算机视觉等，通过大量的数据训练，从中提取和学习特征，最终实现预测、分类、聚类和异常检测等任务。人工智能应用广泛，包括语音助手、人脸识别、无人驾驶等。

互联网+是将互联网与传统行业相结合，涵盖了工业、农业、医疗、教育、金融、旅游等各个行业。利用互联网对传统行业进行产业升级，改造传统产业的生产方式、产业结构等内容，来增强经济发展动力，提升效益。

我认为人工智能和互联网+现在的应用非常广泛，也是未来的趋势和发展方向。

  1. 云计算（Cloud Computing）：云计算是一种以网络为基础的计算服务，可以实现资源共享、数据安全、高效性和可扩展性等优点。
  2. 边缘计算（Edge Computing）：边缘计算是一种分布式计算模型，将计算任务从传统的中央数据中心转移到离数据来源更近的边缘或终端设备上。这样做可以减少网络传输延迟时间和数据流量，提高系统性能和用户体验。

5. 计算机在林业相关领域的应用

  1. 森林资源调查：使用遥感技术和机器学习方法，可以对森林资源进行高效、准确的调查和监测，包括林地类型、树种、树龄、林分结构等信息。
  2. 森林防火：人工智能可以使用图像识别技术，监测森林火灾，提早预警并进行紧急处理。
  3. 森林病虫害管理：人工智能可以帮助识别并监测森林中的病虫害，及时采取措施来防治和控制其扩散。
  4. 林木生产力评估：通过对森林的遥感和机器学习等技术的应用，可以对森林资源的生长模型进行建模，评估森林的生产力。
  5. 林业运筹优化：通过机器学习和优化算法，可以实现森林的生产运作的智能化、自动化管理，提高经济效益。

7. 大学课程跟研究生课程有什么联系
